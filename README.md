A simple text-based RPG combat prototype built in Python.
You face off against the Dragon Lord using spells, mana, and turn-based decisions.

This project was created as part of my early game development and programming journey to practice:

- Game loops
- Player/enemy stats
- Input handling
- Conditional logic
